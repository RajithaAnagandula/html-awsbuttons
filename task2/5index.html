AWS QUETIONS

            21. Which AWS service is used for managing relational databases?
             Amazon RDS
             Amazon DynamoDB
             Amazon Aurora
             Amazon Redshift
        
        22. What is the purpose of Amazon CloudWatch?
         To store and retrieve data objects
         To monitor and manage resources
         To process large datasets
         To distribute content globally
        
        23. Which AWS service provides a managed Kubernetes service?
         Amazon ECS
         Amazon EKS
         Amazon EC2
         Amazon Elastic Beanstalk
        
        24. Which AWS service is used for batch processing of data?
         Amazon Kinesis
         Amazon Redshift
         Amazon Athena
         AWS Batch
        
        25. What is the purpose of AWS VPC?
         To manage virtual machines
         To provide a private network in the AWS cloud
         To monitor network traffic
         To distribute content globally
        
   

Answers:

21. a) Amazon RDS
22. b) To monitor and manage resources
23. b) Amazon EKS
24. d) AWS Batch
25. b) To provide a private network in the AWS cloud
