HTML QUETIONS



21. Which of the following elements is used to create a list with bullet points in HTML?
    a) ul
    b) ol
    c) li
    d) list

22. What does the HTML element iframe stand for?
    a) Inline Frame
    b) Internal Frame
    c) Image Frame
    d) Inline Frame Element

23. Which version of HTML introduced the "canvas" element?
    a) HTML 4
    b) HTML 5
    c) XHTML
    d) HTML+

24. What is the correct HTML for inserting a comment?
    a)  !--This is a comment--
    b) comment This is a comment /comment
    c) comment This is a comment--
    d) //This is a comment//

25. Which tag is used to define metadata in HTML?
    a) metadata
    b) data
    c) meta
    d) information

Answers:
21. a) ul
22. a) Inline Frame
23. b) HTML 5
24. a) !--This is a comment--
25. c) meta
